# IR_A3

## Dependency
    
### Backend Dependency

    pip install flask

### Frontend Dependency

    npm install react-router-dom@6

    npm install axios

    npm install --save @chakra-ui/react

    npm install --save @chakra-ui/icons
    
    import gensim

## Run

### Start Backend Server

    cd src
    
    backend.py

### Start Frontend Server

    cd frontend

    npm start
